BattleShip - Online

/*-------------------------------------------------------------------------*/
/*------------------------------Installation-------------------------------*/
/*-------------------------------------------------------------------------*/

1. Créer deux projets Eclipse : BattleShip-Client et BattleShip-Serveur
2. Créer deux packages dans chaque projet : plateauJoueur et plateauServeur
3. Importer les fichiers que vous trouver dans ce dossier au bon endroit


/*-------------------------------------------------------------------------*/
/*------------------------Lancement de l'application-----------------------*/
/*-------------------------------------------------------------------------*/
1. Lancer la classe : LanceServeur (La console indique si le lancement s'est effectué correctement)
2. Lancer la classe : LanceClient. (Pour simuler plusieurs clients, il est possible de lancer plusieurs fois la classe LanceClient en //)
